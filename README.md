# FCC-Projects 
## My freeCodeCamp.org projects

Projects I completed as part of the freeCodeCamp.org curriculum (https://www.freecodecamp.org/learn/)

* Tribute Page: A simple page with a centered size responsive image and an external link.
* Survey Form: A survey form with different input types (text, number, email, radio and check buttons...) and a dropdown (no JS).
* Product Landing Page: A landing page with a fixed navbar to navigate sections. Also featuring a video tag and an email adress input form.
* Technical Documentation Page: Navigation bar is on the left side of the screen on regular sized screens. The page also features some code tags.
* Personal Portfolio: Single page with three sections and a fixed navbar at the top.
* FtoC: Simple Fahrenheit to Celsius converter. As a bonus, background color changes according to the input temperature.
* Random Quote Machine (React): Gets a random quote from an API and prints it to screen.
* Markdown previewer (React): Takes markdown code written by the user and renders the result in a preview screen area.
* Drum machine (React): Lets user play sounds by clicking on the interface or using the PC keyboard.
* Calculator (React): Simple add/subtract/multiply/divide calculator.
* Countdown Timer (jQuery): Countdown timer with two periods (session + break).
